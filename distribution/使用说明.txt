Git Info Viewer v1.0.0 - 使用说明
========================================

【系统要求】
- Java 8 或更高版本
- Windows 操作系统

【快速启动】
1. 双击运行 "Git-Info-Viewer.bat" 启动应用程序
2. 或者使用命令：java -jar git-info-viewer-1.0.0-jar-with-dependencies.jar

【主要功能】

1. Git仓库浏览
   - 自动发现并显示目录中的Git仓库
   - 查看仓库信息（远程地址、分支、最新提交）
   - 支持分支切换、拉取最新代码

2. Global Search（全局搜索）
   - 关键词搜索：在提交消息中搜索关键词
   - 时间范围搜索：指定开始/结束日期查询
   - 默认显示前100条提交记录
   - 支持结果过滤（按项目、关键词、作者）
   - 点击记录查看详细信息
   - 导出搜索结果到CSV

3. 批量操作
   - 批量切换多个仓库的分支
   - 批量拉取最新代码

【使用提示】
- 左侧目录树：浏览和选择目录
- 右侧面板：查看仓库信息和执行操作
- 勾选复选框：选择要操作的Git仓库
- 双击行：查看仓库详细信息

【常见问题】

Q: 提示"Java is not installed"怎么办？
A: 请先安装Java运行环境(JRE)并配置环境变量

Q: 无法访问Git仓库？
A: 检查网络连接和Git凭据配置

Q: 搜索结果为空？
A: 确保已勾选至少一个Git仓库
