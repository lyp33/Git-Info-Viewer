Git Info Viewer - 使用说明
========================================

系统要求：
- Windows 操作系统
- Java 8 或更高版本

安装说明：
1. 确保已安装 Java 8 或更高版本
2. 将整个文件夹复制到任意位置
3. 双击 start-git-viewer.bat 启动应用程序

文件说明：
- start-git-viewer.bat          启动脚本
- git-info-viewer-1.0.0-jar-with-dependencies.jar    应用程序主文件
- README.txt                    本说明文件

功能特性：
✓ Git仓库信息查看
✓ 分支切换和批量操作
✓ 提交记录搜索
✓ 现代化UI界面
✓ 支持多仓库管理

使用方法：
1. 启动应用程序后，在左侧选择包含Git仓库的目录
2. 右侧会显示该目录下的所有子目录和Git仓库信息
3. 勾选需要操作的Git仓库
4. 使用Switch按钮切换分支或进行批量操作
5. 双击Git仓库可查看详细的提交记录

注意事项：
- 首次启动可能需要几秒钟加载时间
- 确保Git仓库目录有读写权限
- 建议在操作前备份重要代码

如有问题，请检查：
1. Java是否正确安装
2. 文件是否完整
3. 目录权限是否正确
